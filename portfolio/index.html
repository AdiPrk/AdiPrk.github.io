GPU-Driven 6 Million Particle Simulation

My goal for this project was to push the GPU to its absolute limit. I engineered a high-performance simulation pipeline using compute shaders, indirect drawing, sorting, culling, and more optimizations to handle over 6 million particles simultaneously. By rigorously profiling memory access patterns through NVIDIA Nsight, I optimized the system to maintain a stable, playable framerate without creating CPU bottlenecks.
